# 스와스티야 1회수강권 예약 등록 스킬

## 목적

사용자가 스와스티야 1회수강권 예약 정보를 제공하면, 이미 로그인된 Chrome 브라우저 세션으로 스튜디오메이트(`https://swasthya.studiomate.kr`)에 접속해 회원에게 1회수강권을 등록하고 해당 수업에 예약을 등록한다.

## 기본 원칙

- Chrome의 기존 로그인 세션을 사용한다.
- 비밀번호, 인증번호, 결제 정보 등 민감한 정보는 요청하지 않는다.
- 로그인이 풀려 있으면 사용자가 직접 Chrome에서 로그인하도록 안내한 뒤 계속한다.
- 실제 등록, 예약, 취소, 변경처럼 되돌리기 어려운 작업은 실행 직전에 사용자에게 한 번 확인한다.
- 이미 같은 수강권 또는 같은 수업 예약이 존재하면 중복 등록하지 않고 사용자에게 현재 상태를 보고한다.
- 화면에서 확인 가능한 결과를 기준으로 처리하며, 애매한 경우 추측하지 말고 사용자에게 확인한다.

## 사용자에게 받을 정보

예약 처리를 시작하기 전에 아래 정보를 확인한다. 사용자가 이미 일부를 제공했다면 부족한 항목만 질문한다.

- 회원 식별 정보: 휴대폰 전체 번호, 이름
- 수업 정보: 이용일시, 상품명
- 결제 정보: 결제일시


## 처리 흐름

1. Chrome 브라우저에서 `https://swasthya.studiomate.kr`를 연다. 이미 열려있는 탭이 있다면 해당 탭으로 이동한다.
2. 로그인 상태를 확인한다.
3. 회원 메뉴(`https://swasthya.studiomate.kr/users`)에서 제공받은 휴대폰 번호로 회원을 검색한다.
4. 검색 결과가 없다면 오른쪽 하단의 edit 버튼(+ 모양)을 눌러 회원 등록 화면으로 이동하여 이름과 휴대폰번호를 입력하고 "회원 등록 완료" 버튼을 클릭하여 회원을 등록하고, 검색 결과가 존재하면 회원 이름을 클릭하여 회원 상세 화면으로 이동한다.
5. 회원 상세 화면에서 사용중인 수강권을 확인하여 아무것도 등록되어 있지 않은 경우 "새로운 수강권 만들기" 버튼을 눌러 전달받은 상품명으로 검색하여 해당 수강권을 클릭한 후 이용시작일과 이용종료일에 전달받은 이용일시의 일자를 입력하고 "다음" 버튼을 클릭한다. 결제일시에 전달받은 결제일시를 세팅한후 "완료" 버튼을 클릭한다.
6. 사용중인 수강권이 있는 경우에는 더이상 진행하지 말고 해당 수강권이름과 함께 "이미 등록된 수강권이 있습니다."라고 사용자에게 보고한다.
7. 수업 메뉴('https://swasthya.studiomate.kr/lectures') 로 이동하여 전달받은 이용일시의 날짜로 수업목록을 조회하고 이용일시와 동일하거나 가장 가까운 수업일시에 해당하는 수업명을 클릭하여 수업 상세 화면으로 이동한다.
8. 수강회원 영역에 전달받은 휴대폰 번호로 검색하여 방금 수강권을 등록한 회원이 예약 가능한 회원으로 나오면 해당 회원을 선택하여 "예약" 버튼을 클릭한다.
9. 예약 완료 화면 또는 회원 예약 내역에서 등록 결과를 확인한다.
10. 사용자에게 처리 결과를 간단히 보고한다.

## 세부 절차

### 1. 접속 및 로그인 확인

- Chrome에서 스튜디오메이트 사이트를 연다.
- 대시보드, 회원 목록, 수업 일정 등 관리자 화면이 보이면 로그인된 것으로 판단한다.
- 로그인 화면이 보이면 비밀번호를 묻지 말고 “Chrome에서 직접 로그인해 주세요. 완료되면 알려주세요.”라고 안내한다.

### 2. 회원 검색 및 등록

- 회원 메뉴(`https://swasthya.studiomate.kr/users`)로 이동한다.
- 휴대폰 번호로 검색한다. 휴대폰 번호는 회원식별자이기 때문에 동일한 번호로 한개만 등록된다.
- 검색 결과가 0명이면 오른쪽 하단의 edit 버튼(+ 모양)을 눌러 회원 등록 화면으로 이동하여 이름과 휴대폰번호를 입력하고 "회원 등록 완료" 버튼을 클릭하여 회원을 등록한다.

### 3. 1회수강권 등록

- 회원 상세 화면에서 사용중인 수강권 목록을 확인한다.
- 아무것도 등록되어 있지 않은 경우 "새로운 수강권 만들기" 버튼을 눌러 전달받은 상품명으로 검색하여 해당 수강권을 클릭한 후 이용시작일과 이용종료일에 전달받은 이용일시의 일자를 입력하고 "다음" 버튼을 클릭한다. 결제일시에 전달받은 결제일시를 세팅한후 "완료" 버튼을 클릭한다.
- 사용중인 수강권이 있는 경우에는 더이상 진행하지 말고 해당 수강권이름과 함께 "이미 등록된 수강권이 있습니다." 라고 사용자에게 보고한다.

### 4. 수업 예약 등록

- 수업 메뉴('https://swasthya.studiomate.kr/lectures') 로 이동한다.
- 전달받은 이용일시의 날짜로 수업목록을 조회하고 이용일시와 동일하거나 가장 가까운 수업일시에 해당하는 수업명을 클릭하여 수업 상세 화면으로 이동한다.
- 수강회원 영역에 전달받은 휴대폰 번호로 검색하여 방금 수강권을 등록한 회원이 예약 가능한 회원으로 나오면 해당 회원을 선택하여 "예약" 버튼을 클릭한다.

### 5. 결과 확인 및 보고

예약 저장 후 다음 중 하나 이상으로 성공 여부를 확인한다.

- 수업 예약회원 목록에 회원이 표시된다.
- 회원 상세의 이용내역의 예약항목에 해당 수업이 표시된다.
- 1회수강권 잔여 횟수 또는 사용 상태가 올바르게 반영된다.

사용자에게는 다음 형식으로 보고한다.

```text
등록 완료했습니다.
- 회원: {회원명}
- 수강권: 1회수강권 등록/기존 수강권 사용
- 예약: {YYYY-MM-DD} {HH:mm} {수업명}
- 확인: 예약자 목록에서 확인됨
```

## 예외 처리

- 로그인 필요: 사용자가 직접 로그인할 때까지 기다린다.
- 회원 없음: 입력받은 이름/번호와 검색 조건을 알려주고 재확인을 요청한다.
- 동명이인: 화면 정보만으로 특정할 수 없으면 사용자에게 선택을 요청한다.
- 수업 없음: 검색한 날짜, 시간, 수업명을 보고하고 올바른 수업 정보를 다시 요청한다.
- 정원 마감: 예약 가능 여부와 대기 등록 가능 여부를 확인한 뒤 사용자에게 물어본다.
- 이미 예약됨: 중복 예약하지 않고 기존 예약 정보를 보고한다.
- 수강권 등록 실패: 화면의 오류 메시지를 그대로 요약해 사용자에게 알린다.

## 완료 기준

이 스킬은 회원에게 1회수강권이 등록되어 있고, 대상 수업의 예약자 목록 또는 회원 예약 내역에서 예약이 확인되면 완료된 것으로 본다.
